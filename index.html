How it works

One gallery. Separate theme repositories.

Each available theme is maintained in its own repository and can be published independently with GitHub Pages. This collection is the gallery hallway: browse the cards, open demos, inspect source files, and report issues or request additions.

How to use the collection

  2. Open a live demo that matches your style.
  3. Open the repository for the theme you want.
  4. Customize the theme and publish it through GitHub Pages.
